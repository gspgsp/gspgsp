  九万里悟道终归是诗酒田园

<br/>
无题
<br/>
早年同窗始相知，三载瞬逝情却萌。<br />
年少不知愁滋味，犹读红豆生南国。<br />
别离方知相思苦，心田红豆根以生。<br/>
