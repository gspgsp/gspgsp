《无题》

[唐]李商隐

相见时难别亦难，东风无力百花残。<br />
春蚕到死丝方尽，蜡炬成灰泪始干。<br />
晓镜但愁云鬓改，夜吟应觉月光寒。<br />
蓬山此去无多路，青鸟殷勤为探看。
